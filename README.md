# Nationella_Prov-_i-_Svenska_9-
Träningsverktyg inför Nationella Prov i Svenska (Åk 9)  Detta är ett interaktivt verktyg utvecklat för att hjälpa elever att förbereda sig inför de nationella proven i svenska för årskurs 9. Systemet simulerar provets tre delar (Delprov A, B och C) och använder AI-integration för automatisk rättning och poängbedömning i realtid.
